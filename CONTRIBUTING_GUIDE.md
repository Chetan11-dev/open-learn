# Getting Started

## Start Backend

```
cd backend/
yarn
yarn start:dev:db
yarn start:dev
```

## Start Frontend

```
cd frontend/
yarn
yarn start
```

# Tech Stack

Backend: Nestjs, PostgreSQL

Frontend: Reactjs, Redux Toolkit, Tailwindcss
